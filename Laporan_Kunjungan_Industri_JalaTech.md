## 1. Pemahaman Umum Perusahaan

JALA Tech merupakan perusahaan teknologi akuakultur asal Indonesia yang berfokus pada industri budidaya udang. Perusahaan ini menghadirkan solusi digital end-to-end untuk membantu petambak meningkatkan produktivitas, efisiensi operasional, dan keberhasilan panen melalui pemanfaatan data, Internet of Things (IoT), dan kecerdasan buatan (AI).

Layanan JALA Tech mencakup berbagai aspek rantai bisnis budidaya udang, mulai dari pengadaan kebutuhan budidaya melalui JalaSupply, pengelolaan tambak melalui JalaFarm dan aplikasi JALA, dukungan panen melalui JalaHarvest, hingga pemasaran hasil panen melalui JalaMarket. Dengan pendekatan ini, JALA Tech tidak hanya berperan sebagai pengembang aplikasi, tetapi juga sebagai penyedia ekosistem smart aquaculture yang membantu proses budidaya menjadi lebih terukur, modern, dan berkelanjutan.

## 2. Bagaimana Perusahaan Mengolah dan Menganalisis Data

JALA Tech menjadikan data sebagai pusat pengambilan keputusan bisnis dan operasional. Data yang berasal dari sensor tambak, catatan aktivitas budidaya, data pakan, angka kematian, hingga transaksi keuangan dikumpulkan ke dalam sistem big data yang disebut JALA Cloud untuk dipantau dan dianalisis lebih lanjut.

Data tersebut kemudian dimanfaatkan untuk berbagai kebutuhan, seperti memantau kondisi tambak, menilai kelayakan kredit, menjaga traceability atau ketertelusuran produk udang, serta membantu petambak mengambil keputusan yang lebih cepat dan akurat. Dalam analisis lanjutan, JALA Tech menggunakan pendekatan data science dan machine learning untuk memprediksi survival rate udang, memprediksi kualitas air, dan mendeteksi potensi penyakit berbahaya pada udang.

Beberapa algoritma yang disebut digunakan dalam proses analisis ini antara lain Random Forest, Gradient Boosting, XGBoost, serta stacked model yang pada evaluasi internal dilaporkan memberikan performa terbaik untuk prediksi survival rate. Selain itu, JALA Tech juga memanfaatkan computer vision berbasis YOLO untuk camera-based sampling dan OCR berbasis TensorFlow untuk mengubah catatan manual petambak menjadi data digital.

## 3. Bagaimana Perusahaan Memanfaatkan IoT untuk Akuisisi Data

Pemanfaatan IoT di JALA Tech terlihat jelas melalui perangkat bernama Baruni yang digunakan untuk melakukan akuisisi data langsung dari tambak secara real-time. Perangkat ini dapat mengukur beberapa parameter penting kualitas air sekaligus, yaitu pH, suhu, salinitas, oksigen terlarut (DO), dan ORP.

Data dari Baruni menjadi sumber informasi utama dalam pemantauan kondisi tambak karena kualitas air sangat menentukan kesehatan dan pertumbuhan udang. Dengan sistem IoT ini, proses pengumpulan data tidak lagi bergantung sepenuhnya pada pencatatan manual, melainkan dilakukan secara lebih cepat, konsisten, dan akurat sehingga risiko kesalahan pengamatan dapat ditekan.

Integrasi IoT dengan platform digital JALA memungkinkan data sensor dikirim, disimpan, dan dianalisis secara berkelanjutan. Hasilnya, petambak dapat memperoleh peringatan dini dan dasar rekomendasi tindakan yang lebih tepat untuk menjaga kualitas budidaya dan mengurangi potensi gagal panen.

## 4. Bagaimana Perusahaan Mengimplementasikan Python untuk Mendukung Bisnis

Python memiliki peran penting dalam ekosistem teknologi JALA Tech karena digunakan dalam pengembangan analisis data dan model machine learning perusahaan. Pada laporan kunjungan dijelaskan bahwa proses pengolahan data dan kecerdasan buatan di JALA Tech banyak melibatkan Python dengan library standar industri seperti Pandas, Scikit-learn, dan NumPy.

Melalui Python, JALA Tech dapat membangun pipeline analisis data untuk membaca, membersihkan, mengolah, dan memodelkan data budidaya udang dalam skala besar. Python juga mendukung pembuatan model prediksi survival rate, prediksi kualitas air, klasifikasi risiko penyakit, hingga implementasi computer vision dan OCR yang berkontribusi langsung pada efisiensi operasional perusahaan.

Dari sisi bisnis, implementasi Python memberi nilai tambah karena membantu perusahaan menghasilkan insight yang dapat digunakan untuk pengambilan keputusan, otomatisasi proses analitik, dan pengembangan layanan digital berbasis data. Dengan dukungan infrastruktur cloud seperti AWS Lambda, solusi berbasis Python yang dikembangkan JALA Tech dapat dijalankan secara efisien untuk mendukung kebutuhan bisnis yang dinamis.
